== Description ==

O plugin @MediaPost - Formulário de cadastro permite aos usuários da ferramenta de e-mail marketing @MediaPost criarem formulários de cadastro integrados à suas contas com rapidez e facilidade.

Caso não seja cliente, [crie sua conta gratuitamente](https://www.mediapost.com.br/criar-conta?como_conheceu=Plugin%20Wordpress) e confira todos os recursos desta poderosa ferramenta de e-mail marketing.

Com este plugin você poderá criar diversos formulários de cadastro integrado com suas listas de contatos e inserí-los em qualquer lugar de seu site ou blog do wordpress.

Com os recursos de Widget e Shortcodes você poderá inserir formulários:

 - no cabeçalho das páginas e artigos
 - no rodapé do site/blog
 - nas laterais do seu blog
 - no conteúdo de páginas e blogs
 - enfim, em qualquer lugar de seu site wordpress.

== Installation ==

Instale este plugin, extraindo os arquivos baixados e faça o upload deles para o diretório /wp-content/plugins/ de seu wordpress.

Feito isso, ative o plugin através do menu "Plugins".

Após instalado, precisará inserir as chaves de integração de sua conta @MediaPost para integrar o plugin com sua conta.

= Como conseguir as chaves de integração de minha conta @MediaPost? =

Para usar este plugin, precisará solicitar à equipe de suporte @MediaPost as chaves de acesso à sua conta.

Para isso, siga os seguintes passos:

 - [Acesse sua conta @MediaPost](https://www.mediapost.com.br/login)
 - No menu superior, clique em Suporte
 - Depois clique em Meus chamados
 - Clique no botão Novo chamado
 - Preencha os campos escolhendo a Categoria "Chaves de integração"
 
A equipe de suporte @MediaPost irá gerar suas chaves e enviá-las através do chamado aberto em até 24 horas úteis.

== Frequently Asked Questions ==

= Este plugin é grátis? =

Sim. Este plugin não possui nenhuma funcionalidade paga. Ele é 100% gratuito.

= Preciso conhecer HTML para usar o plugin? =

Não. Este plugin dispõe de modelos de formulários prontos com as opções de mudar cores e textos dos mesmos.

= Entendo de HTML, posso criar meu próprio formulário usando códigos? =

Sim. Caso não queira usar um dos modelos prontos. Poderá criar seu formulário de cadastro em HTML. Você terá liberdade para mudar tudo nele!

= Posso criar uma conta @MediaPost para teste? =

Sim. Você pode criar uma conta gratuita para conhecer a ferramenta de e-mail marketing [clicando neste link](https://www.mediapost.com.br/criar-conta?como_conheceu=Plugin%20Wordpress).

= O plugin funciona com contas testes e pagas? =

Sim. O plugin irá funcionar com qualquer conta @MediaPost. Não precisa pagar para isso.
